BxSlider - Custom Version

The popular open source responsive slider plugin BxSlider has been modified to accept an unlimitted number of break points. Support for animated layers has also been added.

  • Added data attributes support
  • Added break points support
  • Added animatable layers
  • Added automatic slide width calculation
  • Added self calling capability

Simple Codeless Setup

Use the data attribute data-call="bxslider" together with the .bxslider class to quickly setup a slider without writing any piece of code! Remember to wrap your .bx-slider class with a parent class so that you can style the slider. For usage in your markup refer to the HTML tab.

Adding Options

Use the data attribute data-options="" to pass any bxSlider supported options. The options should be wrapped inside {} and seperated by commas. View HTML tab for usage.

Layers Setup

To add animatable layers use the .bx-layer on any element contained in your slide. Now just the following data attributers to your layer and your good to go.

  • •Set data-anim="" with one of the animation classes contained in the animate.css library
  • •Set data-dur="" with the duration of your animation in milliseconds
  • •Set data-delay="" with the delay of your animation in milliseconds

Positioning Layers

There are no premade classes for position the layers. You need to position the layers either using your own classes or using inline styles.

Adding Break Points

Use the data attribute data-breaks="" to pass options targeting specific screen break points. Each break point is defined as a set of options enclosed in curly brackets. All of BxSlider's options are supported.. View HTML tab for usage.

Alternative Setup

If you prefer not to use data attributes, the breaks and other new options can still be passed in the bxslider() call. View HTML tab for usage.

$(document).ready( function() {

$('.my-slider').bxSlider({

slideMargin: 5,

autoReload: true,

breaks: [{screen:0, slides:1, pager:false},{screen:460, slides:2},{screen:768,slides:3}]

});

});
