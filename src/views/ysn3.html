Animations Library

Information on Usage, styles and extra classes.

Don't need this module?

To disable any module simply uncomment it's import statement in uikit.scss. This will generate a smaller stylesheet without the module.

Based on the Animate.css Library

Animate.css is a bunch of cool, fun, and cross-browser animations for you to use in your projects. Great for emphasis, home pages, sliders, and general just-add-water-awesomeness. The official project can be found here github.com/daneden/animate.css

Usage

To use simply add the class .animated to an element, along with any of the animation names. That's it! You've got a CSS animated element. Super!

"animated bounceIn" src="images/demo/logo-light.png">

Usage in BxSlider

To add animatable layers use the .bx-layer on any element contained in your slide. Now just the following data attributers to your layer and your good to go.

  • •Set data-anim="" with one of the animation classes contained in the animate.css library
  • •Set data-dur="" with the duration of your animation in milliseconds
  • •Set data-delay="" with the delay of your animation in milliseconds

Animations only work on this BxSlider Version!

This theme uses a customized version of BxSlider. The shown examples won't work on other versions!
